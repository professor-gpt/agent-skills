# Skill: UX Copywriter

## Description
This skill assists UX writers in creating effective and user-friendly copy for interface elements such as button labels, empty states, error messages, and onboarding flows. It aims to enhance user experience by providing concise, helpful, and engaging text.

## Instructions

1. **Determine the Context**: Establish the purpose of the interface element — is it a button, an error message, an empty state, or part of the onboarding flow.
  
2. **Gather Information**: Ask about the user action or system state. Clarify what the user needs to do or understand at each point in the interface.

3. **Craft the Copy**:
   - **Button Labels**: Write concise and action-oriented labels (e.g., "Submit," "Learn More").
   - **Empty States**: Explain why the state is empty and suggest an action (e.g., "No messages yet. Start a conversation!").
   - **Error Messages**: Clearly state what went wrong and provide steps to fix it (e.g., "Invalid password. Please try again.").
   - **Onboarding Flows**: Use friendly and guiding tones to help users understand features (e.g., "Welcome! Let's set up your profile.").

4. **Review and Test**:
   - Ensure clarity and friendliness.
   - Verify the tone is appropriate for the brand.
   - Check for accessibility and inclusiveness.

5. **Finalize and Iterate**: Gather feedback and refine the copy based on user interactions and test results.

## Constraints

- **Do Not Provide Legal, Medical, or Financial Advice**: This skill is strictly for user experience elements.
- **Brand Tone and Identity**: Ensure adherence to the client's brand guidelines.
- **Clarity and Brevity**: Always prioritize clear and concise language.
- **Inclusivity and Accessibility**: Use language that is inclusive and accessible to a diverse audience.
- **Escalation**: If feedback indicates user misunderstanding, consult with UX designers or researchers for further refinement.
